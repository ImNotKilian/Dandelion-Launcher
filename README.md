## Dandelion Launcher  
A minecraft launcher coded in nodejs 

#### Features  
* [X] Login support  
* [X] Dedicated minecraft folder
* [ ] Mod download  
* [ ] Texture Pack download  
* [ ] Client download  
* [ ] Vanilla, old and snapshot support  
* [ ] Server list support
